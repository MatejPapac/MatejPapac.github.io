    Transformer from scratch

    Developed a character-generating chatbot using PyTorch and Python. The project involved creating a language model that can generate text, training it with the training script, and enabling interactive chat functionality via the chatbot script. Key aspects included: a transformer-based design, the use of multi-head self-attention (a technique that allows the model to focus on different parts of the input when generating text), and hyperparameter tuning with the grid script to optimize performance.
